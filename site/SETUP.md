# Pumkin site

The marketing + docs site for Pumkin, built with [Astro](https://astro.build)
and deployed on Vercel.

## Develop

```
npm install
npm run dev
```

## Build

```
npm run build
```

Output goes to `dist/`. Vercel builds and deploys automatically on push to
`main`.

The site is static plus a few prerendered pages — there's no database, no
payment provider, and no environment variables are required to build it.
